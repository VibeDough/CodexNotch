# Contributing to CodexNotch

Thanks for helping improve CodexNotch.

## Workflow

1. Fork the repository.
2. Create a focused branch for one change.
3. Keep the change small and avoid unrelated formatting or refactoring.
4. Run the test suite:

   ```sh
   DEVELOPER_DIR=/Applications/Xcode.app/Contents/Developer swift test --jobs 2 -Xswiftc -warnings-as-errors
   ```

5. Open a pull request that explains the problem, the change, and how it was verified.
