# CodexNotch 0.1.4

这次更新补全了空闲悬停面板中央的留白，让每日使用节奏更直观，同时保持原来的紧凑尺寸。

## 新功能

- 空闲悬停时显示连续使用天数和今日使用强度。
- 增加 14 天活跃进度条，快速了解近期使用节奏。
- 点击“今日节奏”区域可直接打开本地 Token 日报。

## 体验优化

- 新内容不增加刘海面板高度，也不挤压左右两侧的今日用量、套餐与版本信息。
- 继续复用固定顶部画布和统一状态动画，任务、确认、设置与日报切换不会重新移动窗口顶部。
- 所有统计继续仅从本机 Codex 日志读取，不上传任务或用量数据。

> 当前仍是 Apple Silicon 预览版，尚未经过 Apple 公证。首次启动请右键应用并选择“打开”。
