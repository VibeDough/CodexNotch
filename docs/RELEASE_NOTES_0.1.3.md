# CodexNotch 0.1.3

这次更新让 CodexNotch 从状态提示继续向轻量工作入口前进，同时保持所有数据只在本机处理。

## 新功能

- 本地最近任务搜索，可按任务、项目与摘要返回对应 Codex 对话。
- 智能拖入：网页、GitHub Skill、图片与普通文件显示不同的下一步操作。
- 本地 Token 日报、14 天圆环热力、近 7 日能量轨迹与个人基线评价。
- 日报显示连续使用天数和 14 天活跃进度；高强度日期打开时播放一次轻量星点庆祝。
- 新完成任务使用“圆环绘制 → 对勾回弹”的一次性确认动画。
- 可设置的日报提醒，以及额度首次低于 50% 时的一次性提醒。
- 每日自动检查 GitHub Release，也可在设置中手动检查。
- 设置改为“通用 / 功能 / 关于”横向分类，并加入官网、GitHub、作者与版本入口。

## 修复与体验优化

- 重启后先恢复最近一次有效额度，减少短暂显示 0 的情况。
- 新增状态沿用固定顶部画布和统一 0.22 秒过渡，避免尺寸切换时出现裂隙或黑框。
- 设置页由纵向长列表改为固定高度分类菜单。
- 正式构建移除日报内部预览入口。

> 当前仍是 Apple Silicon 预览版，尚未经过 Apple 公证。首次启动请右键应用并选择“打开”。
