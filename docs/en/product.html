49agent Notch

A lightweight macOS status layer for Codex, designed around the MacBook notch.

Requirements

  • macOS 14 or later
  • Apple silicon
  • Codex installed and signed in locally

What it shows

  • Running, analyzing, confirmation, completion, and connection states
  • Task title, model, reasoning level, token usage, and elapsed time
  • Remaining usage, today's tokens, reset time, and Codex version
  • Files, URLs, and text prepared for a new Codex conversation

Install

Download the latest DMG. Drag the app into Applications. The preview is not Apple-notarized, so right-click the app and choose Open on first launch.

Language

The website is available in Chinese and English. The current app interface is Chinese; full in-app English localization is planned separately.
