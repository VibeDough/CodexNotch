CodexNotch

我是 Henry恒宇。做 CodexNotch，是因为把任务交给 Codex 后，我总会切去做别的事，却又忍不住回来确认进度。

它把真正需要知道的状态放在 MacBook 刘海旁：任务是否还在运行、是否正在等你，以及什么时候可以回去看结果。它不是另一块工作台，只是一层安静、随时可返回 Codex 的状态。

日常怎么用

  • 运行任务时继续留在浏览器、文档或设计工具里。
  • 从刘海旁查看任务标题、耗时、模型、推理强度和 Token。
  • 需要输入或确认时获得提醒，点击即可回到对应 Codex 对话。
  • 空闲时靠近刘海，查看今日消耗、剩余用量、重置时间与版本。
  • 把文件、网址或文字拖进去，准备一条新的 Codex 对话。

安装

下载 Apple 芯片版 DMG，拖入“应用程序”即可。需要 macOS 14+ 与本机 Codex。

当前仍是预览版，尚未经过 Apple 公证。首次启动请右键应用并选择“打开”。

留在本机

CodexNotch 读取本机 Codex 会话与桌面日志来判断状态，不上传对话内容、任务信息或用量数据。拖入本地文件时传递的是文件路径和分析提示，文件不会由 CodexNotch 上传。

现在仍有限制

状态同步依赖 Codex 当前的本地日志格式；Codex 更新后可能需要继续适配。预览版目前仅提供 Apple Silicon 构建，任务跳转和文件读取也会受到 Codex 本身权限的影响。
