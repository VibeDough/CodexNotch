功能演示视频指南

建议录制 60–90 秒、16:9、60fps 的实机演示。核心叙事是：离开 Codex 窗口后，任务进度仍然留在视线顶部。

推荐镜头

  1. 空闲状态与实体刘海。
  2. 启动任务后切换到浏览器。
  3. 展开并收回多个任务。
  4. 拖入文件并在 Codex 新建对话。
  5. 查看用量和重置时间。
  6. 点击完成提示返回对应任务。

录制重点

  • 每个状态停留至少 1.5 秒。
  • 不要在展开动画中间剪切。
  • 隐藏通知、私人路径与未公开项目。
  • 如果出现裂隙、错位或掉帧，重新录制镜头。
